#关于消费者向提供者发送方法信息并返回执行结果的这个过程，Demo中的RPC的传输方式提供了两种，一种是基于http协议的传输，在代码中使用了一个内嵌的tomcat来实现通过http传输数据，另一种则是基于Netty框架，通过Tcp协议(套接字)来传输信息。
